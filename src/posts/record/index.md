# 记录

有些日子过去得很快，如果不写下来，很多细节就像被风吹散一样，再想找回来已经很难。

记录不是为了证明生活有多特别，而是为了提醒自己：普通的一天也值得被看见。今天路过的街、听到的一句话、突然冒出来的想法，都可能在以后变成重新理解自己的线索。

我想把这些片段留下来。它们不一定完整，也不一定成熟，但是真实。真实的东西有重量，哪怕只是一段很短的文字，也能让某个时刻停一停。
